# Poker Bankroll Managment

Web based poker bankroll management app.

## Server setup

1. env file with following variables
   - PORT
   - MONGO_URL
   - JWT_SECRET
   - SENDGRID_API_KEY
   - FROM_EMAIL
   - NODE_ENV
   - CLIENT_URL

## Client setup

1. env file with following variables
   - VITE_BASE_URL
   - VITE_ASSETS_URL
